The Challenge

Motion designers needed to export chapter markers and cue points from After Effects for use in interactive players, subtitle systems, and audio editing tools. Conversely, they wanted to drive AE animations from external data sources without manual keyframing.

My Role

Sole developer of the ExtendScript toolkit, including the export panel UI, JSON schema design, and expression templates for data-driven animations using AE's native footage().sourceData functionality.

Technical Deep Dive

ExtendScript: Marker Export
// Extract markers from selected composition
function exportMarkersToJSON(comp) {
    var markers = comp.markerProperty;
    var output = {
        projectName: app.project.file.name,
        composition: comp.name,
        frameRate: comp.frameRate,
        markers: []
    };
    
    for (var i = 1; i <= markers.numKeys; i++) {
        var mv = markers.keyValue(i);
        output.markers.push({
            time: markers.keyTime(i),
            comment: mv.comment,
            chapter: mv.chapter,
            cuePoint: mv.cuePointName !== "",
            url: mv.url,
            params: mv.getParameters()
        });
    }
    
    return JSON.stringify(output, null, 2);
}
